# ClangFormat for Windows
Repository to produce statically linked x64 binary of [clang-format](https://clang.llvm.org/docs/ClangFormat.html)<br>.

# How to download
[Releases page](https://github.com/PolarGoose/clang-format-for-Windows/releases)

# How to build
Run `build.ps1` script. You need to have Visual Studio installed
